GameFo Android Application:
This app is intended to be a player guide to the Bloodborne video game.
Currently it only displays the information about locations in the game.
The info for the guide is scraped form a website using Jsoup.
All the red email buttons will take you back to the previouse screen. 
Use Android Studio to run the app on either an emulator in device.
Works on APK 14:Android 4.00 or later.
